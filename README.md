# 🧠 LLM & Advanced RAG Repository

A comprehensive collection of **Large Language Model (LLM)** and **Retrieval-Augmented Generation (RAG)** notebooks demonstrating modern AI workflows using **Qwen**, **Pydantic**, **Sentence Transformers**, and advanced retrieval techniques.

---

## 📌 Overview

This repository provides end-to-end implementations of LLM applications, structured output generation, vector databases, hybrid retrieval, reranking, and custom embedding model fine-tuning.

It is designed for AI Engineers, Data Scientists, and researchers who want to build production-ready RAG pipelines.
